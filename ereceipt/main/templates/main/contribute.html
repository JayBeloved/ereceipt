Renaming Course Materials

1. 

If it is a course material the name should start with "mat"

2.

If there are different materials for different topics in a single course, the naming should follow: mat-course code - topic e.g. mat-acc-101-double-entry

3.

If there's only one material for the course, the naming should follow: course code - course title e.g. mat-bus-204-business-communication

Examples

1. mat-acc-202-departmental-accounts
2. mat-bkf-201-investment-appraisal

Renaming Past Questions

1. 

If it is a past questions the name should start with "pq"

2.

The order for naming past questions is : pq - course code - session end

Note

If the session is 2021/2022, session end will just be 22. If the session is 2019/2020, then session end will be 20...

Examples

1. pq-acc-302-22
2. pq-bkf-201-21 
3. pq-gst-309-20
